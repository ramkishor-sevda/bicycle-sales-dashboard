#  Sales Data Dashboard (Excel Project)

This repository showcases an interactive **Sales Data Dashboard** built in Microsoft Excel.  
It uses **pivot tables, charts, slicers, and sunburst visualizations** to provide clear insights into sales performance across different customer segments, products, and regions.

---

##  Dataset Overview

The dataset contains **21 columns** including:

| Column Name | Description |
|------------|-------------|
| Segment | Customer segment category |
| Customer_ID / Customer_Name | Unique customer identifiers |
| Product_ID / Product_Name | Details of products sold |
| Units Sold | Quantity sold |
| Sale Price / Manufacturing Price | Pricing and cost information |
| Gross Sales | Total revenue before discount |
| Discounts | Discount applied to sales |
| Sales | Final sales after discount |
| COGS | Cost of goods sold |
| Profit & Profit (000) | Profit earned per transaction |
| Date, Month, Quarter, Year | Time-based analysis fields |
| Country | Customer location |

---

##  Dashboard Features

✔ Total Sales, Profit & Units Sold Summary  
✔ Sales by **Segment**  
✔ Sales by **Country**  
✔ Sales Trend over Months / Years  
✔ Product-wise Performance  
✔ Profit Contribution Sunburst Chart  
✔ Interactive Slicers for drill-down analysis  

---

##  Sheets Included

| Sheet Name | Purpose |
|-----------|---------|
| `Sales Data` | Raw dataset used for analysis |

- Enterprise & Government segments drive the highest revenue.
- Profit contribution varies significantly across product categories.
- Seasonal sales trends appear across different quarters.
